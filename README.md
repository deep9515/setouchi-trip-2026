# Setouchi Trip Site 2026

Private travel board for the 2026 Sunrise Seto / Naoshima / Kagawa trip.

## Local Preview

Open `index.html` directly, or run:

```sh
python3 -m http.server 4174
```

Then open `http://127.0.0.1:4174/`.

## Privacy Note

This repository is intended to stay private.

GitHub Pages is not a reliable private website for a normal personal account: a Pages URL may be publicly reachable even when the repository is private, depending on the account/organization plan and Pages access control settings. Use a real access-controlled host for a private URL.
